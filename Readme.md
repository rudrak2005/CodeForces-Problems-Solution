This README contains **350 Codeforces problems** arranged from beginner-friendly implementation to advanced competitive programming topics.

- Competitive programming fundamentals

## 🎯 Practice Rules

1. Start by solving **1 problem every day**.
2. Think independently for **30–45 minutes before reading the editorial**.
3. Write a **dry run** for the example.
4. Implement the solution independently in **C++17**.
5. After getting accepted, note the **Pattern + Mistake + What You Learned**.
6. After completing a problem, change `[ ]` to `[x]`.
7. Revise important problems again.
